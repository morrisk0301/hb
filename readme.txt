디지털 청진음 Validation Version 1.0 06/25/2018

환경설정
--------------------------------------------------------------------------
- Python 버전 3.0이상(3.6 권장)
- Tensorflow 버전 1.8.0
- Librosa 버전 0.6.0
- Numpy 버전 1.14.2

Tensorflow, Librosa, Numpy의 경우는 PIP를 사용해 Install
Anaconda를 이용한 설치를 권장합니다

※버전이 다를 경우 실행이 되지 않을 수 있으니, 
가급적 위의 버전을 이용해 주시기 바랍니다
--------------------------------------------------------------------------

사용법
--------------------------------------------------------------------------
1. 학습된 Model을 model폴더에 추가

2. 심음 .wav 파일을 sounds폴더에 추가

3. Validation.py 열기

4. 변수 filename, modelname 설정

5. 실행
